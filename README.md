# AI Document Assistant

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/VVSSPRAVEEN/AI-agent-project.git
   cd AI-agent-project

2.**Create a Virtual Environment**
python -m venv env
source env/bin/activate  # On Windows use `env\Scripts\activate`


3.**Install Dependencies**

pip install -r requirements.txt

4.**Run the Application**
streamlit run main.py

**Tools Used**
Python 3.8+: Versatile language with rich library support.
Streamlit: Simple, interactive web app for data applications.
FAISS: Efficient similarity search and clustering of dense vectors.
PyMuPDF: PDF parsing and handling.
Requests: Simplified HTTP requests for API integration.
